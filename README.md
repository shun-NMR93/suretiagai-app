# Surechigai（すれちがい）

任天堂の「すれ違い通信」に着想を得た、BLE（Bluetooth Low Energy）を利用したiOS向けすれ違い通信アプリです。

近くにいるユーザー同士がバックグラウンドでもプロフィールを交換し、出会った履歴を記録できます。

---

# 主な機能

## プロフィール

* 初回起動時のプロフィール登録
* ニックネーム
* ひとこと
* 出身都道府県
* オリジナルアバター（デフォルメ狐）
* プロフィール編集
* 端末へのプロフィール保存

---

## ホーム画面

* Core Motionを利用した歩数計測
* 今日の歩数表示
* 1,000歩ごとの進捗バー
* BLE通信状態を表すレーダー・波紋アニメーション
* ストリートパス風のグラデーションデザイン

---

## すれ違い通信

* BLEによるプロフィール交換
* フォアグラウンド通信
* バックグラウンド通信
* ロック画面中のプロフィール交換
* 受信プロフィールの保存
* すれ違った時刻の記録
* 「今さっき」「○分前」「○時間前」などの表示

---

## すれ違い履歴

* 交換したプロフィール一覧
* 交換日時表示
* 都道府県ごとのコレクション表示

---

# 技術

* SwiftUI
* CoreBluetooth
* CoreMotion
* UserDefaults
* JSON
* Background BLE Communication

---

# 開発環境

* Xcode 16.4+
* iOS 18.0+
* SwiftUI

---

# 起動方法

1. `Surechigai.xcodeproj` をXcodeで開く
2. `Surechigai` ターゲットを選択
3. 実機で実行（BLE通信は実機のみ対応）
